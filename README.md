# Sys Probe
Sys Probe is a tool to help gather infomation about a system and the operating system. This infomation can help people troubleshoot a issue.
## Supported Plaforms
Sys Probe is currently only supported on Windows, but there are plans to support other platforms as well (such as Linux)
